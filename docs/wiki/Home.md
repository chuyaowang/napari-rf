# Welcome to the napari-rf Wiki

`napari-rf` is a specialized napari plugin designed for interactive image segmentation using Random Forest (RF) classifiers. It bridges the gap between manual annotation and automated analysis by allowing researchers to train classifiers directly within the napari viewer.

## Core Capabilities
- **Interactive Training**: Draw labels on your images and train a classifier in real-time.
- **Multiscale Feature Extraction**: Automatically generates a rich feature set (intensity, edges, texture) using `scikit-image`.
- **Probability Mapping**: Outputs class probabilities, allowing for soft segmentation and uncertainty analysis.
- **Persistence**: Save and load trained classifiers (`.joblib`) for reproducible analysis across different datasets.
